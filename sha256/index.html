SHA-256 Basics

What this tool can do

  • Hash text to SHA-256 (hex or Base64)
  • Hash files in the browser (ArrayBuffer)
  • Verify by comparing computed vs. expected digest
  • HMAC-SHA-256 (hash with a secret key)

All hashing happens locally using the Web Crypto API‚Äînothing is uploaded.

Inputs

  • Text: UTF-8 encoded (emoji & non-Latin supported)
  • Files: drag & drop or file picker (size limited by memory)
  • Output: hex (default) or Base64

About SHA-256

  • Secure hash from the SHA-2 family (256-bit output)
  • Designed for integrity (not encryption)
  • Do not use plain SHA-256 for password storage‚Äîuse Argon2/bcrypt/PBKDF2 instead

Hash vs. HMAC

  • Hash: digest of data only (public, anyone can recompute)
  • HMAC: digest using a secret key (authenticates source & integrity)

How to use

  1. Choose Text or File mode.
  2. Enter text or drop a file; select output as Hex or Base64.
  3. Click Hash to compute the SHA-256 digest locally.
  4. (Optional) Enter an Expected hash to auto-verify.
  5. (Optional) For HMAC-SHA-256, provide a secret key and compute.

Tips

  • Whitespace & newlines matter‚Äîhashes change if the text changes even slightly.
  • Large files may need time/memory; keep the tab focused for best performance.
  • To compare from terminals, use shasum -a 256 file (macOS) or sha256sum file (Linux).

SHA-256 Best Practices

üßÇ Don‚Äôt hash passwords directly

Use Argon2, bcrypt, or PBKDF2 with salt & proper parameters.

üîê Prefer HMAC for authenticity

Plain hashes don‚Äôt prove who produced the data‚ÄîHMAC with a secret key does.

üìè Encode consistently

Always hash the same byte representation (UTF-8). Mixed encodings lead to mismatches.

üßÆ Show both hex & Base64

Hex is common in CLIs; Base64 is compact for UIs/APIs.

Technical Implementation

Text ‚Üí SHA-256 (hex/Base64)

const enc = new TextEncoder();

async function sha256Text(text, out = "hex") {
  const bytes = enc.encode(text);
  const digest = await crypto.subtle.digest("SHA-256", bytes);
  const buf = new Uint8Array(digest);
  return out === "base64" ? toBase64(buf) : toHex(buf);
}

function toHex(bytes) {
  return [...bytes].map(b => b.toString(16).padStart(2, "0")).join("");
}

function toBase64(bytes) {
  let bin = "";
  bytes.forEach(b => bin += String.fromCharCode(b));
  return btoa(bin);
}

File ‚Üí SHA-256

async function sha256File(file, out = "hex") {
  const buf = await file.arrayBuffer(); // reads whole file into memory
  const digest = await crypto.subtle.digest("SHA-256", buf);
  const bytes = new Uint8Array(digest);
  return out === "base64" ? toBase64(bytes) : toHex(bytes);
}
// Note: For very large files, consider a streaming/incremental WASM hash library.

Verify against expected

async function verifyTextSHA256(text, expectedHexOrB64) {
  const expected = expectedHexOrB64.trim().toLowerCase();
  const actualHex = (await sha256Text(text, "hex")).toLowerCase();
  if (expected.includes("=") || /[+/]/.test(expected)) {
    // Treat as base64
    const actualB64 = await sha256Text(text, "base64");
    return actualB64 === expectedHexOrB64;
  }
  return actualHex === expected;
}

HMAC-SHA-256 (sign/verify)

async function hmacKeyFromText(secret) {
  return crypto.subtle.importKey(
    "raw", enc.encode(secret),
    { name: "HMAC", hash: "SHA-256" },
    false, ["sign", "verify"]
  );
}

async function hmacSha256(message, secret, out = "hex") {
  const key = await hmacKeyFromText(secret);
  const sig = await crypto.subtle.sign("HMAC", key, enc.encode(message));
  const bytes = new Uint8Array(sig);
  return out === "base64" ? toBase64(bytes) : toHex(bytes);
}

async function verifyHmacSha256(message, secret, signatureHex) {
  const key = await hmacKeyFromText(secret);
  const sigBytes = hexToBytes(signatureHex);
  return crypto.subtle.verify("HMAC", key, sigBytes, enc.encode(message));
}

function hexToBytes(hex) {
  if (hex.length % 2) throw new Error("Invalid hex");
  const out = new Uint8Array(hex.length / 2);
  for (let i = 0; i < out.length; i++) {
    out[i] = parseInt(hex.substr(i*2, 2), 16);
  }
  return out;
}
