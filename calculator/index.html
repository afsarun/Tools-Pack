Allowed: digits, + − × ÷ * / ( ) . and % (as percent). Use parentheses for order of operations.

Use cases

  • Shopping & discounts: work out sale prices in seconds (e.g., 25% * 79.99 → how much you save).
  • Budgeting & bills: split dinner, rent, or utilities fairly (try (125.60 + 89.40) / 3).
  • School & homework: double-check arithmetic before you submit.
  • Forms & admin: sanity-check totals and percentages so mistakes don’t slip through.

How to use

  1. Type your math. Example: (12 + 7) / 3 * 2.
  2. Use % for percentages. Example: 12% * 200 → 24 (we treat 12% as 12/100).
  3. Press Enter to calculate. Use Shift+Enter for a new line.
  4. Click Copy result to copy the answer, or Clear to start again.
  5. You can type × and ÷ if you like — we read them as * and /.

Tips

  • Works offline after the first load.
  • Browsers use floating-point math, so you might see tiny rounding quirks (e.g., 0.1 + 0.2).
  • This is a basic calculator — no functions like sin, sqrt, etc.

FAQ

Does this calculator work offline?

Yes. After the first load, everything runs in your browser.

Are my calculations tracked?

No. We don’t send your input anywhere — it stays on your device.

How does % work here?

A number followed by % is a percentage of 1. For example, 12% becomes 12/100, so 12% * 200 = 24.

Which operations are supported?

Basic arithmetic with parentheses: + − × ÷ (also * and /), decimals, and %.

Why do I sometimes see long decimals?

That’s normal with floating-point math in browsers. You may see tiny rounding artifacts (e.g., 0.30000000000000004).

Is this mobile-friendly?

Yes. It works in any modern mobile or desktop browser.

Does it support scientific functions?

Not in this tool — it keeps things simple. For advanced math, use a scientific calculator.
