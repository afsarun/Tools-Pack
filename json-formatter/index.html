How to use

  1. Paste or type JSON into Input.
  2. Click Format to pretty print (2-space indent) and show the result in Output.
  3. Click Minify for compact one-line JSON.
  4. Click Validate to check correctness without producing output.
  5. Use Copy Output to copy the result to your clipboard.
