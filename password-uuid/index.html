How to use

  1. Choose a Length with the slider (16–128).
  2. Toggle the character sets you want: a–z, A–Z, 0–9, and symbols.
  3. Click Generate to create a new random password.
  4. Click Copy to copy it to your clipboard.
  5. Regenerate as needed until you’re satisfied.

Security tips

  • Prefer 20+ characters for important accounts.
  • Use a password manager to store passwords securely.
  • If a site blocks certain symbols, disable symbols and regenerate.

FAQ

Are passwords stored or sent to a server?

No. Everything runs in your browser only. We do not send or store your passwords on any server.

Is this generator crypto-secure?

Yes. It uses the Web Crypto API (window.crypto.getRandomValues) for high-quality randomness.

What length should I use?

We recommend 16+ characters, and 20+ for sensitive accounts. Longer is stronger.

Why include symbols?

Symbols increase the character set size and entropy, making passwords harder to guess. Some sites may restrict certain symbols—toggle them off if needed.

Does it work offline?

Yes. Once loaded, the generator continues to work without internet.
