কীভাবে ব্যবহার করবেন

  1. উপরের বক্সে টেক্সট লিখুন (বাংলা/ইমোজি সহ যেকোনো ইউনিকোড টেক্সট চলবে)।
  2. এনকোড চাপলে Base64 আউটপুট পাবেন; ডিকোড চাপলে Base64 ইনপুট থেকে টেক্সটে রূপান্তর হবে।
  3. ক্লিয়ার দিয়ে ইনপুট/আউটপুট রিসেট করুন।

টিপস

  • এই টুল TextEncoder/TextDecoder ব্যবহার করে—তাই ইউনিকোড-সেফ।
  • URL-safe দরকার হলে আউটপুটে +→-, /→_ ব্যবহার করা যায়; প্রয়োজনে প্যাডিং = বাদ দিতে পারেন।
  • খুব বড় টেক্সটে ব্রাউজারের মেমোরি সীমাবদ্ধতা প্রযোজ্য।

প্রশ্নোত্তর/faq

হ্যাঁ। UTF-8 বাইটে রূপান্তর করে Base64 এনকোড/ডিকোড করা হয়।

+ → -, / → _ এবং প্রয়োজনে শেষে থাকা = সরিয়ে নিন।

না—সবকিছু কেবল আপনার ব্রাউজারেই প্রক্রিয়াকরণ হয়; কোনো ট্র্যাকিং নেই।

ইনপুটে অবৈধ ক্যারেক্টার/প্যাডিং থাকতে পারে। স্পেস/লাইন ব্রেক সরিয়ে আবার চেষ্টা করুন।

এই টুল টেক্সটের জন্য। ফাইলের Base64 প্রয়োজন হলে ব্রাউজারের File API দিয়ে করা যায়—ভবিষ্যৎ আপডেটে যুক্ত হতে পারে।
