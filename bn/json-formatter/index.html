কীভাবে ব্যবহার করবেন

  1. উপরে JSON পেস্ট/টাইপ করুন।
  2. ফরম্যাট দিলে সুন্দর ইন্ডেন্টেড আউটপুট পাবেন।
  3. মিনিফাই দিলে স্পেস/লাইনব্রেক বাদ দিয়ে ছোট সাইজ।
  4. ভ্যালিডেট JSON সিনট্যাক্স ঠিক আছে কি না দেখায়।
  5. আউটপুট কপি দিয়ে ফলাফল ক্লিপবোর্ডে কপি করুন।
  6. Ctrl/⌘ + Enter চাপলে দ্রুত ফরম্যাট হবে।
