How to Use Quick Notes

  1. Start typing in the box — your text auto-saves after you pause.
  2. Click Export .txt to download your notes as a text file.
  3. Use Import to load a .txt file back into the editor.
  4. Click Clear to remove everything (you’ll be asked to confirm).

Tips

  • Works offline after the first load.
  • Privacy: localStorage is not encrypted — avoid highly sensitive data.
  • Size limit: localStorage is typically ~5 MB per site; split very large notes into multiple files.
