Quick Notes Basics

What this tool can do

  • Fast note taking in your browser (no account)
  • Auto-save notes to localStorage as you type
  • Multiple notes with titles & timestamps
  • Search & pin important notes
  • Export/Import notes as JSON or text

Everything is stored locally on your device—nothing is uploaded.

Storage & privacy

  • Location: Browser localStorage
  • Scope: Per browser & device (no sync)
  • Limits: Typically ~5–10 MB per origin

Note features

  • Markdown-friendly text area (plain text stored)
  • Pin/star notes to keep them on top
  • Word/character count (optional)
  • Trash/restore (soft delete) option

What counts as “local-only”?

  • Notes never leave your browser unless you export them
  • Clearing site data or using private windows may remove notes
  • Switching browsers/devices won’t carry notes over automatically

How to use

  1. Click New and give your note a title.
  2. Start typing in the editor—your note auto-saves every few keystrokes.
  3. Use Search to filter by title or content.
  4. Click Pin to keep a note at the top; use Trash to archive (reversible).
  5. Use Export to download your notes; Import to load them back.

Tips

  • For long-term backup, export regularly (JSON). Keep a copy in your cloud drive.
  • Pinned notes are ideal for TODOs or daily logs.
  • Use headings (e.g., # Title) and lists for readability; the app stores plain text, so it’s future-proof.

Technical Implementation

Data model

const NOTE_KEY = "quick-notes:v1";
type Note = {
  id: string,         // uuid
  title: string,
  body: string,
  pinned: boolean,
  deleted: boolean,   // soft delete
  createdAt: number,  // epoch ms
  updatedAt: number   // epoch ms
};

Load & save

function loadNotes() {
  try {
    return JSON.parse(localStorage.getItem(NOTE_KEY)) || [];
  } catch {
    return [];
  }
}

function saveNotes(notes) {
  localStorage.setItem(NOTE_KEY, JSON.stringify(notes));
}

Create/update (auto-save)

function upsertNote(notes, partial) {
  const now = Date.now();
  let list = [...notes];
  const i = list.findIndex(n => n.id === partial.id);
  if (i >= 0) {
    list[i] = { ...list[i], ...partial, updatedAt: now };
  } else {
    list.push({
      id: crypto.randomUUID(),
      title: partial.title || "Untitled",
      body: partial.body || "",
      pinned: false,
      deleted: false,
      createdAt: now,
      updatedAt: now
    });
  }
  saveNotes(list);
  return list;
}

Search & sort

function filterNotes(notes, q = "") {
  const s = q.trim().toLowerCase();
  const kept = notes.filter(n => !n.deleted && 
    (n.title.toLowerCase().includes(s) || n.body.toLowerCase().includes(s)));
  return kept.sort((a, b) => (b.pinned - a.pinned) || (b.updatedAt - a.updatedAt));
}

Export / Import

function exportNotes(notes) {
  const blob = new Blob([JSON.stringify(notes, null, 2)], { type: "application/json" });
  const a = document.createElement("a");
  a.href = URL.createObjectURL(blob);
  a.download = "quick-notes.json";
  a.click();
  URL.revokeObjectURL(a.href);
}

async function importNotes(file) {
  const text = await file.text();
  const notes = JSON.parse(text);
  if (!Array.isArray(notes)) throw new Error("Invalid notes file");
  saveNotes(notes);
  return notes;
}

Soft delete & restore

function trashNote(notes, id, deleted = true) {
  return upsertNote(notes, { id, deleted });
}

function pinNote(notes, id, pinned = true) {
  return upsertNote(notes, { id, pinned });
}
